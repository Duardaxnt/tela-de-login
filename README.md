# Tela de Login

Projeto de uma tela de login simples usando HTML, CSS e JavaScript.

## Como usar

1. **Clonar o repositório**:
   ```bash
   git clone https://github.com/Duardaxnt/tela-de-login.git
Abrir o arquivo index.html no navegador para visualizar a tela de login.

Estrutura do Projeto
index.html: Arquivo principal HTML contendo a estrutura da página de login.

style.css: Arquivo de estilo CSS com as definições de layout e design da página.

script.js: Arquivo JavaScript responsável pela lógica de validação do formulário de login.

Funcionalidades
Formulário de login com campos para nome de usuário e senha.

Validação básica dos campos de entrada.

Mensagens de erro amigáveis em caso de dados inválidos.

Tecnologias Utilizadas
HTML5: Para a marcação da estrutura da página.

CSS3: Para o design e layout responsivo.

JavaScript: Para a lógica e interatividade da página.
